Sobre Mim

Trabalho com PHP desde junho de 2023, começando como estagiário e, a partir de julho de 2025, atuando como profissional PJ. Atualmente, sou responsável por ajustar e aprimorar sistemas internos, criando relatórios, gráficos e exportações para CSV e Excel, além de gerar QR Codes para URLs conforme a necessidade.

Desenvolvo seguindo o padrão MVC, estruturando Model, View e Controller para garantir organização e escalabilidade no código. Também trabalho com Ajax, JavaScript, jQuery e CSS para construir interfaces dinâmicas e responsivas.

Tenho experiência em manipulação de bancos de dados, inserindo e recuperando informações para alimentar as funcionalidades mencionadas. Além disso, utilizo JSON para integração e verificação de APIs externas, garantindo a troca de dados eficiente entre sistemas.

Sou apaixonado por código limpo, funcional e com boa usabilidade, sempre buscando soluções que agreguem valor ao negócio e facilitem o trabalho da equipe.
